# 🔍 DevFinder — GitHub User Explorer

Search any GitHub username and see their profile, top repos, star counts, language breakdown, and more.

## Features
- 🔍 Real-time GitHub user search
- 📁 Top repos with stars, forks, and language badges
- 🌙 Dark / Light mode
- ⚡ Async with Promise.all — parallel API calls
- 🖥️ Demo mode — shows Linus Torvalds by default

## Tech Stack
- React 18 · Vite · GitHub REST API · CSS-in-JS

## Getting Started
```bash
git clone https://github.com/yourusername/devfinder.git
cd devfinder && npm install && npm run dev
```
